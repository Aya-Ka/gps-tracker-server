# gps-tracker-server
A GPS tracker API server

### Prerequisites
node.js (https://nodejs.org/en/)

npm (https://www.npmjs.com/)

### Installing
```
npm install
```

### Lanuching API server
```
npm start
```

### Swagger URL
```
$host/documentation
```

### Testing
```
npm test
```

### Linter
```
npm run lint
```
