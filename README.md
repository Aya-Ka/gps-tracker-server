# gps-tracker-server
A GPS tracker API server

### Prerequisites
node.js (https://nodejs.org/en/)

npm (https://www.npmjs.com/)

### Lanuching API server without Docker
```
npm install
```
```
npm start
```

### Lanuching API server Docker
```
docker-compose up
```

### Swagger URL
```
$host/documentation
```

### Testing
```
npm test
```

### Linter
```
npm run lint
```
